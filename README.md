🇪🇬 Egipto: Ingenieria de la Legumbre - Smart Contract (Base)
Este es el primer contrato inteligente de la coleccion "Sabores de Africa". A diferencia de otras gastronomias, la cocina egipcia es una tecnologia de gestion del tiempo sobre el grano seco, fundamentada en el ciclo del Nilo y el uso de grasas vegetales ancestrales.

🛠️ Arquitectura Tecnica del ADN Egipcio
Cada registro en este contrato valida los pilares que definen la cocina del Valle del Nilo:

Horas de Remojo (uint256): La variable critica. Registra la activacion enzimatica y suavizado de legumbres (ej. 24h para el Ful, 12h para el Koshary).

Matriz de Lipidos (string): Clasifica el uso de Samna (mantequilla clarificada) para platos tradicionales frente a aceites de semillas para el consumo diario.

Sello de Identidad (Dukka/Shatta): Validacion booleana del uso de mezclas de semillas y escala inmutable de picante (1-5).

📖 Ejemplos Registrados On-Chain

Ful Medames: 24h de remojo, Aceite de Oliva/Samna. El pilar proteico del desayuno.

Koshary: 12h de remojo, Aceite vegetal. La sincronizacion perfecta de carbohidratos.

Ta'ameya: 24h de remojo, Aceite de girasol. La variante egipcia del falafel usando habas.
